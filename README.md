# 🏥 Hospital Resource Allocation & Demand Forecasting

## 📌 Overview
This project analyzes hospital patient demand and resource allocation using data analytics and forecasting techniques.

## 🎯 Objectives
- Analyze demand trends
- Compare demand vs admissions
- Identify high-demand services
- Forecast future demand

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- Power BI
- Excel

## 📊 Key Features
- Demand trend analysis
- Service-wise demand distribution
- Acceptance & refusal rate calculation
- Forecasting using moving average
- Interactive Power BI dashboard

## 📈 Key Insights
- Demand exceeds admissions in most cases
- Emergency services have highest demand
- High refusal rate indicates capacity issues
- Utilization rate ~93%
